바로 매칭하기 왜 빠를까요?

핵심 특징

번개 매칭

회원가입 없이 즉시 대기열 → 연결.

안전 보안

hCaptcha + 욕설/스팸 필터 + 도배 차단.

가벼운 설계

DB 없이 인메모리 매칭. 빠른 응답성.

모바일 최적화

앱 같은 경험. 입력창 고정, 큰 터치영역.

사용 방법

  1. “바로 시작”을 눌러 앱으로 이동
  2. 캡차 인증을 통과하고 대기열 등록
  3. 상대가 연결되면 채팅 시작
  4. 언제든 다음/종료 가능

자주 묻는 질문

익명인가요?

예. 닉네임 외 개인정보를 저장하지 않습니다.

봇/스팸은 어떻게 막나요?

hCaptcha와 금칙어/도배 방지 로직으로 차단합니다.

모바일에서도 잘 되나요?

네. 모바일 퍼스트 반응형입니다.
